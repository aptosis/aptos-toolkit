Expand description

Helper trait to convert string-like references to &UncasedStr.

Example

use uncased::AsUncased;

let string = "Hello, world!".as_uncased();
assert_eq!(string, "hello, world!");
assert_eq!(string, "HELLO, world!");
assert_eq!(string, "HELLO, WORLD!");
